### Usage

To Install:   
`npm install eser`

To validate with ESLint:   
`eslint --config ./node_modules/eser/.eslintrc.json ./src/`

To validate with CSScomb:   
`csscomb --lint --config ./node_modules/eser/.csscomb.json ./src/`
